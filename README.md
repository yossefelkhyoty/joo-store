# Joo Store 🛒

Welcome to **Joo Store**, a modern e-commerce front-end application built with React and Vite.

## 🌟 Live Demo

Check out the live application here: [Joo Store Live Demo](https://joo-store-one.vercel.app/)

## 🛠️ Tech Stack

- **React 19**
- **React Router DOM v7** (for client-side routing)
- **Vite** (for fast build and development)
- **Vanilla CSS** (for responsive and custom styling)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository
   ```sh
   git clone <repository-url>
   ```
2. Navigate into the project directory
   ```sh
   cd my-app
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## 📜 Available Scripts

In the project directory, you can run:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run preview`: Locally preview the production build.
- `npm run deploy`: Deploys the project to GitHub Pages.
